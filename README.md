# calcular-corrente-de-circuitos
programa feito com o intuito de auxiliar no cálculo de correntes em circuitos por malhas.
após a identificação das malhas do circuito, chega-se nos valores das correntes de I1, I2, I3, etc. 
para chegar ao resultado final das correntes é necessário criar uma matriz quadrada com os valores das correntes nas malhas
e outra matriz com o valor de igualdade das malhas, esta tendo apenas uma coluna
é preciso apenas alterar o valor das matrizes e rodar que o programa mostrará o valor das correntes nas malhas
